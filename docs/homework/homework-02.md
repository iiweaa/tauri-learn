## 作业提示

### 配置文件位置

- `src-tauri/tauri.conf.json` - TAURI主配置文件
- `src-tauri/Cargo.toml` - Rust依赖配置
- `src-tauri/capabilities/default.json` - 权限配置
- `package.json` - Node.js依赖配置
- `vite.config.js` - Vite构建配置

### 常用配置项

**窗口配置**：
```json
{
  "title": "窗口标题",
  "width": 1200,
  "height": 800,
  "minWidth": 500,
  "minHeight": 400,
  "center": true
}
```

**权限配置**：
```json
{
  "permissions": [
    "core:default",
    "opener:default",
    "fs:allow-read-file"
  ]
}
```

### 注意事项

1. **JSON格式**：确保JSON格式正确，注意逗号和引号
2. **重启应用**：修改配置后需要重启应用才能生效
3. **权限匹配**：添加的权限必须与使用的API匹配
4. **identifier唯一性**：确保identifier格式正确且唯一
