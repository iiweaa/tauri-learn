# 第1课作业：TAURI框架简介与环境搭建

## 作业说明

完成本作业以巩固第一课的学习内容。作业分为**必做部分**和**选做部分**，请至少完成必做部分。

## 作业要求

### 必做作业（必须全部完成）

#### 任务1：环境搭建验证
- [ ] 成功安装Rust开发环境（运行 `rustc --version` 和 `cargo --version` 验证）
- [ ] 成功安装Node.js（运行 `node --version` 验证）
- [ ] 成功创建TAURI项目并运行（`npm run tauri dev` 能正常启动应用）

#### 任务2：修改窗口配置
- [ ] 在 `tauri.conf.json` 中修改窗口标题为"我的第一个TAURI应用"
- [ ] 将窗口大小改为 1000x700
- [ ] 验证修改生效（重启应用后能看到新的标题和窗口大小）

#### 任务3：创建计算器命令
- [ ] 在 `src-tauri/src/main.rs` 中创建一个 `calculate` 命令
  - 命令接收两个数字参数（`a` 和 `b`）
  - 返回两个数字的和
- [ ] 在前端（`src/App.jsx` 或相应文件）中调用这个命令
- [ ] 添加输入框让用户输入两个数字
- [ ] 添加按钮触发计算
- [ ] 显示计算结果

#### 任务4：界面美化
- [ ] 使用CSS美化应用界面
- [ ] 至少包含以下样式：
  - 居中的布局
  - 美观的按钮样式（有hover效果）
  - 输入框样式
  - 合适的字体和颜色搭配
  - 适当的内边距和外边距

### 选做作业（加分项）

#### 任务5：错误处理（选做）
- [ ] 在Rust的 `calculate` 命令中添加错误处理
  - 处理除零错误（如果后续添加除法功能）
  - 处理无效输入
- [ ] 在前端捕获并显示错误信息
- [ ] 使用友好的错误提示UI

#### 任务6：扩展计算功能（选做）
- [ ] 添加减法、乘法、除法功能
- [ ] 使用下拉菜单或按钮组让用户选择运算类型
- [ ] 根据选择的运算类型执行相应的计算
